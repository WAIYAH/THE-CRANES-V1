## Kanunka FC - The Cranes Website
A modern, responsive website for Kanunka FC - The Cranes, built with passion for football and cutting-edge web technologies.


## 📖 Overview
This project is a fully functional front-end website for Kanunka FC - The Cranes, a fictional football club. The website serves as a digital hub for fans, players, and stakeholders, providing information about the club, matches, players, news, and career opportunities. It is built using HTML5, CSS3, Bootstrap 5, and vanilla JavaScript, with a focus on modularity, responsiveness, and user experience.

# The website includes the following core pages:

Home Page: Hero section, club motto, latest news, and upcoming matches.
Matches Page: League table, fixtures, results, and statistics (to be implemented).
Players Page: Player profiles, filters, and Player of the Month (to be implemented).
Blog Page: News, match reviews, and interviews with category filters.
Contact Us Page: Contact form, club information, and social media links.
Gallery Page: Photo and video gallery with lightbox and embedded YouTube videos.
About Us Page: Club history, management, staff, and legends.
Careers/Vacancies Page: Job listings, application form, and player trials.

## 🚀 Features
Responsive Design: Built with Bootstrap 5 to ensure compatibility across mobile, tablet, and desktop devices.
Dark/Light Mode: Toggle between dark and light themes for better accessibility and user preference.
Smooth Animations: GSAP for transitions and AOS for scroll animations to enhance user experience.
Modular Code: Each page has its own CSS and JavaScript file for maintainability.

## Interactive UI:
Category filters for blog posts and gallery photos.
Lightbox effect for gallery images.
Client-side form validation for contact and career applications.
Global Navbar & Footer: Consistent navigation and footer across all pages, with hover effects and animations.
Social Media Integration: Links to Twitter, Facebook, Instagram, and YouTube in the footer and Contact Us page.

## 🏁 Getting Started
Prerequisites
A modern web browser (e.g., Chrome, Firefox, Edge).
A code editor like Visual Studio Code.
Basic knowledge of HTML, CSS, and JavaScript.

## Installation
Clone the Repository:
git clone https://github.com/waiyah/kanunka-fc-website.git

Alternatively, download the ZIP file and extract it.
Navigate to the Project Directory:
cd kanunka-fc-website
Open the Project:
Open the project folder in your code editor.
To view the website, open any HTML file (e.g., contact.html) in a web browser. You can do this by:
Double-clicking the file, or
Using a local server (recommended for better performance):

# If you have Python installed
python -m http.server 8000
Then, navigate to http://localhost:8000 in your browser.

## Dependencies
The project uses CDN links for external libraries, so no additional installation is required. The following are included:
Bootstrap 5 (CSS and JS)
GSAP
AOS

## 🖥️ Usage
Navigation: Use the global navbar to navigate between pages (Home, Matches, Players, Blog, Contact Us, Team, Gallery, About Us, Careers).
Dark/Light Mode: Click the moon/sun icon in the navbar to toggle between themes.
Forms:
Contact Form: Fill out the form on the Contact Us page to send inquiries (client-side validation included).
Career Application Form: Apply for jobs or player trials on the Careers page (client-side validation included).
Gallery: Click on images to view them in a lightbox; watch embedded YouTube videos.
Blog: Filter posts by category (Match Analysis, Transfers, Club News).
